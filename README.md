# Watch List — NgRx Store vs Signals

A vibe coding test project exploring NgRx Store vs Angular Signals for state management in a simple watch list app.

## Stack

- Angular 21 (standalone, OnPush)
- NgRx Signal Store (movies feature)
- NgRx Store (TV shows feature)
- Vitest

## Dev

```bash
npm start   # http://localhost:4200
npm test
npm run build
```
